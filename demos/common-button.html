Has icon, 有图标的 Way 1, use slot and custom source (recommended), 第一种方式, 使用 slot 和自定义的源 (推荐) This way can reduce network spending. 该方法可减少网络开销 Also, you have more customize on this way, like use another icon set (in iconify), or use image as a icon. 同时, 你可以拥有更多的自定义空间, 例如使用另一个图标集 (在 iconify 内), 或是拿图片当作图标使用.
Way 2, use attribute, 第二种方法, 使用属性 The value will be in md-icon, use iconLib:iconName to use iconify-icon, use iconName to use Material Symbols or custom by CSS variable --md-icon-font by icon font.

Accessibility, 无障碍功能 Use data-aria-label attribute to custom aria-label. 使用 data-aria-label 属性来自定义 aria-label. Support all the other ARIA attributes for button, see details at the following table. 其他用于按钮的 ARIA 属性全部支持, 详见下表.
Attributes, 属性 Types:
  • single choice means choose one of all names
  • toggle means give a attribute with no content for true, do not give this attribute for false
Name, 名称 Type, 类型 Default value, 默认值 Description, 描述
(filled) / tonal / elevated / outlined / text single choice (filled) Controls button's style
disabled toggle false If true, common button will be disabled. Controls disabled, aria-disabled
data-role string button Controls role
data-aria-label string Controls aria-label
data-aria-haspopup string Controls aria-haspopup
data-aria-controls string Controls aria-controls
data-aria-expanded boolean Controls aria-expanded
data-aria-selected boolean Controls aria-selected
label string Controls label content, the default value of aria-label
leading string Controls leading icon name
trailing string Controls trailing icon name
tag string Controls custom inner element tag name
