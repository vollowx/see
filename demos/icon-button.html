Attributes, 属性

Types:

  • single choice means choose one of all names
  • toggle means give a attribute with no content for true, do not give this attribute for false
Name, 名称 Type, 类型 Default value, 默认值 Description, 描述
(filled) / tonal / outlined / standard single choice (filled) Controls icon button's style
disabled toggle false If true, icon button will be disabled. Controls disabled, aria-disabled
data-role string button Controls role
data-aria-label string Controls aria-label
data-aria-haspopup string Controls aria-haspopup
data-aria-controls string Controls aria-controls
data-aria-expanded boolean Controls aria-expanded
data-aria-selected boolean Controls aria-selected
icon string Controls icon name (web font), not recommended
toggle toggle false If true, icon button will have 2 states - unselected and selected. Controls selected
