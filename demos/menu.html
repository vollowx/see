Accessibility, 无障碍功能 Supported keys 支持的按键 a-z / A-Z for focusing listitem by first char,
ArrowUp for focusing previous listitem,
ArrowDown for focusing next listitem,
PageUp / Home for focusing first listitem,
PageDown / End for focusing last listitem,
Enter / Whitespace (' ') for activating listitem,
 Attributes, 属性 Types:
  • single choice means choose one of all names
  • toggle means give a attribute with no content for true, do not give this attribute for false
Name, 名称 Type, 类型 Default value, 默认值 Description, 描述
anchor-el string (id name) Controls menu's controller, must have value
(auto) / normal / dense toggle (normal) Controls menuitem's size. As usual, height = 48px on touch device, height = 36px on others, this attribute override it
anchor-origin {"v": string, "h": string} ({"v": "bottom", "h": "left"}) Controls menu's anchor origin, demo
transform-origin {"v": string, "h": string} ({"v": "top", "h": "left"}) Controls menu's transform origin, demo
